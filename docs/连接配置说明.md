## 📝 功能说明

### 支持的对话模式

1. **普通模式** (默认)
   - 一次性返回完整的 AI 回复
   - 适合短文本回复

2. **流式模式** (可选)
   - 逐字返回 AI 回复
   - 更好的用户体验
   - 在 `App.tsx` 中修改 `USE_STREAMING = true` 启用

### API 接口说明

#### 1. 健康检查
```
GET /
响应：{"status": "ok", "message": "AI Chat API is running"}
```

#### 2. 普通聊天
```
POST /api/chat
请求体：{
  "messages": [
    {"role": "user", "content": "你好"}
  ]
}
响应：{
  "message": "AI 回复内容",
  "error": null
}
```

#### 3. 流式聊天
```
POST /api/chat/stream
请求体：同普通聊天
响应：SSE 流式数据
格式：data: {"content": "文本片段"}
```

## ⚠️ 常见问题

### 1. 端口被占用
**错误：** `Address already in use`
**解决：** 
- 关闭占用端口的程序
- 或修改配置使用其他端口

### 2. 前端无法连接后端
**检查：**
- 后端是否已启动（端口 8000）
- 前端代理配置是否正确
- 防火墙是否阻止连接

### 3. API 调用失败
**检查：**
- API Key 是否有效
- 网络连接是否正常
- 查看后端控制台的错误日志
