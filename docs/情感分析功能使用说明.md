# 情感分析功能使用指南

## 📋 概述

已成功将 **cntext** 情感分析模块集成到你的 AI Chat 系统中。现在你的 LLM 可以使用两种方法进行情感分析：

1. **词典法** - 基于传统情感词典，快速高效
2. **LLM法** - 基于大模型深度理解，更准确

---

## 🚀 快速开始

### 方法一：在聊天中直接使用

启动后端服务后，直接在聊天界面输入：

```
帮我分析这句话的情感：这个产品非常好用，我很满意！
```

或者：

```
分析这段文字的情绪：我今天很伤心，因为考试没考好
```

AI 会自动调用情感分析工具并返回结果。

---

## 🔧 技术细节

### 新增的工具函数

#### 1. `analyze_sentiment(text, method)` - 情感倾向分析

**参数：**
- `text`: 待分析的文本内容
- `method`: 分析方法
  - `"dictionary"` - 词典法（推荐，快速）
  - `"llm"` - 大模型法（更准确，需要 Ollama）

**返回示例：**
```
📊 情感分析结果（词典法）:

文本：这个产品非常好用，我很满意！

情感倾向：正面 😊
情感得分：0.75（范围：-1到1）
正面词汇数：3
负面词汇数：1
总词汇数：12
```

#### 2. `analyze_emotion(text)` - 细粒度情绪分析

**参数：**
- `text`: 待分析的文本内容

**识别的情绪类型：**
- 开心 😊
- 愤怒 😠
- 悲伤 😢
- 惊讶 😲
- 厌恶 🤢
- 恐惧 😨
- 中性 😐

**返回示例：**
```
🎭 情绪分析结果:

文本：我今天很开心，因为收到了礼物

主要情绪：开心 😊
置信度：0.92（范围：0到1）
```

## 💡 使用建议

### 何时使用词典法？
✅ 批量处理大量文本  
✅ 需要快速响应  
✅ 无需联网环境  
✅ 对精度要求不高  

### 何时使用 LLM 法？
✅ 需要高精度分析  
✅ 理解复杂语境  
✅ 分析讽刺、反语等  
✅ 需要细粒度情绪分类

## 🌟 高级用法

### 1. 自定义情感词典

```python
import cntext as ct

# 加载特定领域词典
finance_dict = ct.read_yaml_dict('zh_common_FinanceSenti.yaml')
result = ct.sentiment(text="股价上涨", diction=finance_dict)
```

### 2. 批量情感分析

```python
texts = ["好评", "差评", "一般"]
for text in texts:
    result = analyze_sentiment(text, method="dictionary")
    print(result)
```

### 3. 结合其他工具使用

```python
# 先搜索，再分析情感
search_result = tavily_search("某产品评价")
sentiment_result = analyze_sentiment(search_result, method="dictionary")
```

## ❓ 常见问题

### Q1: 提示 "cntext 模块未找到"？
**A:** 确保路径正确，检查 `AIchattest/情感分析模块test/cntext-main/` 是否存在。

### Q2: LLM 方法报错？
**A:** 需要先启动 Ollama 服务：
```bash
ollama serve
ollama pull qwen2.5:3b
```

### Q3: 如何提高分析精度？
**A:** 
- 使用 LLM 方法而非词典法
- 选择更适合领域的词典
- 预处理文本（去除噪声）

### Q4: 支持英文吗？
**A:** 支持！设置 `lang='english'` 即可：
```python
result = ct.sentiment(text="I love this product", 
                     diction=dict, 
                     lang='english')
```
